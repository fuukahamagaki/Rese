# Rese
飲食店の予約サービスです。店舗一覧や店舗詳細、飲食店のご予約などの利用が可能です。

## 目的
外部の飲食店予約サービスは手数料を取られるので自社で予約サービスを持ちたいため作成。

## 機能一覧
新規登録・ログイン・ログアウト機能
マイページ（予約一覧、お気に入り一覧）
飲食店一覧
飲食店詳細
予約機能
お気に入り機能
エリア、ジャンル、店名検索

##　 使用技術
Laravel 8.*
